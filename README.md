# Spellcheck
A simple spellchecker written in python. 

## Usage

When you run the program it will look like this:

```
Enter the text you want to spellcheck:
>> The quick brown fox jumps over the lazy dog


Capital Word --------> The
Good Word    --------> quick
Good Word    --------> brown
Good Word    --------> fox
jumpz --------> jump
jumpz --------> jumps
Good Word    --------> over
Good Word    --------> the
Good Word    --------> lazy
Good Word    --------> dog


#########################
##       RESULTS        
##  %88 Correct
#########################
```
